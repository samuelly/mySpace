2. swift是强类型语言，每种变量都有自己的类型，但是swift有一个强大的特性叫做类型推导(type inference),可以通过上下文来自动推断出变量的类型。
eg: let digit = sender.currentTitle    (这里的digit自动推断为string类型)


3.  某些变量类型后面的? 表示一种类型：Optional
	Optional: this type has only 2 values , one value is 'not set' (未设)，一般如果设为nil的话就认为这是‘未设’的
			another value is 'something' (有值)
